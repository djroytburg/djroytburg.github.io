Glossary

cruxan experiment both sides agree would change their mind
advocate A / Ban agent arguing one of the two explanations
minimal-crux gatethe check that the two sides actually predict DIFFERENT outcomes — if not, the test is useless
self-preference (SPB)how much a model favors its own output above a coin-flip (0.50)
τ_lo / τ_hithe pre-agreed thresholds that split the result into A-wins / undecided / B-wins
degeneratea ‘test’ where both sides predict the same outcome, so it can decide nothing
indeterminatethe result landed in the middle band — the experiment couldn't call it
